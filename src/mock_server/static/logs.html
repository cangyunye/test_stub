使用说明

Mock Server — HTTP/WebSocket 桩服务管理平台。

快速开始

  1. 在端点管理创建 HTTP 端点或 WebSocket 通道
  2. 使用复制按钮获取 curl / Python / 浏览器控制台示例
  3. 调用 mock 端点返回配置的响应

认证方式

  • stub-x-token 请求头 — 值为用户名，用于多租户隔离
  • 公共端点无需 token，归属端点需要匹配的 token

模板变量

响应体支持 Jinja2 模板：{{ "{{ headers.Host }}" }}、{{ "{{ body }}" }}、{{ "{{ query.key }}" }}
